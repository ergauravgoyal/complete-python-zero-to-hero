# complete-python-zero-to-hero

# Python Bootcamp

A workspace for learning Python.

## Setup

1. **Activate the virtual environment**

   ```bash
   source venv/bin/activate
   ```

2. **Install packages** (when needed)

   ```bash
   pip install <package-name>
   ```

## Getting Started

Open the Jupyter notebooks in this folder to work through the exercises. Start with `numbers.ipynb` to explore Python basics.
